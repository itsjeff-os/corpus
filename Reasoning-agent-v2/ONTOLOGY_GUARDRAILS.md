# Ontology guardrails

## Permanent substrate

Reasoning graph is permanent. Expansion depth is adaptive. Visible epistemics are minimal unless useful.

## Core chain

```text
observation -> interpretation -> inference -> conclusion -> feedback
```

## Terms

### Reasoning
The structural substrate that separates observations, interpretations, inferences, conclusions, and feedback.

### Epistemics
The status of what the assistant thinks it knows: stance, confidence, supporting signals, contradictions, and user feedback.

### Metacognition
The assistant's operating state: current mode, ambiguity tolerance, reflection depth, action permission state, cognitive snapshots, and self-evaluation.

### Governance
Rules and mechanisms that prevent assumed consent, unauthorized action, excessive autonomy, and silent ontology drift.

### Memory
Conversational and temporal continuity, especially user/project context over time. Memory is not the same as inference truth.

### Knowledge
Source-grounded facts, documents, entities, and provenance.

### Constraints
Slowly weighted behavioral patterns created from supported inference, confirmation, or repeated interaction evidence. Soft constraints are not hard commands.

### Strategies
Reusable response/action patterns selected by the cognitive router.
