# Custom GPT setup (optional)

If you want stronger separation (one agent per GPT), create 4 Custom GPTs:
- Automation Engineer (AE)
- Lighting Designer (LD)
- DevOps Smart Home (DO)
- Life Architect (LA)

Paste the corresponding file from `gpts/` into each GPT’s Instructions.

You can optionally add Actions using the OpenAPI schemas in `actions/`.

Note: Custom GPTs are created via the ChatGPT GPT editor and live outside Projects.
